# CodeReviewEnv — Design System: The Celestial Terminal

> Generated by **Stitch** (Obsidian Synth theme) and implemented with Tailwind CSS v4 + React.

---

## 1. Creative North Star

**"The Celestial Terminal"** — A high-fidelity, dark-mode-first environment that mirrors the precision of elite engineering. The interface is not a series of boxes, but a sophisticated, multi-layered IDE leveraging the depth of the deep navy cosmos and the energetic pulse of Indigo and Pink.

## 3. Typography

- **Font**: Inter (Google Fonts)
- **Code Font**: JetBrains Mono
- **Display (3.5rem)**: Hero text, tight tracking (-0.02em)
- **Headline (1.5rem)**: Section headers, weight 600
- **Body (0.875rem)**: Regular content, line-height 1.6
- **Label (0.75rem)**: Uppercase, tracking +0.05em, used for metadata

---

## 4. Glassmorphism Rules

- **Glass Card**: `rgba(23, 23, 47, 0.6)` + `backdrop-blur(16px)` + ghost border
- **Glass Header**: `rgba(12, 12, 32, 0.8)` + `backdrop-blur(20px)`
- **Glass Sidebar**: Solid `surface-container-low` (#111127)
- **Hover**: Border transitions to `primary/25`, ambient glow appears

---

## 5. Component Patterns

### Buttons
- **Primary**: Gradient `primary` → `primary-dim` at 135°, no border
- **Secondary**: Ghost border, transparent bg
- **Tertiary**: Text-only, primary color, hover shows surface-bright bg

### Severity Badges
- **Critical**: `error/10` bg, `error` text
- **Major**: `warning/10` bg, `warning` text
- **Minor**: `primary/10` bg, `primary` text
- **Nit**: `surface-container-high` bg, `outline` text

### Difficulty Badges
- **Easy**: `success/10` bg, `success` text, `success/20` border
- **Medium**: `warning/10` bg, `warning` text, `warning/20` border
- **Hard**: `error/10` bg, `error` text, `error/20` border

---

## 6. Spacing Scale

Base unit: 4px. Key values: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64.

---

## 7. Animation Guidelines

- **Transitions**: 200-300ms ease
- **Entrance**: `fadeInUp` with stagger (0.08s intervals)
- **Hover**: `translateY(-1px)` + glow shadow
- **Sidebar**: Spring animation (stiffness 380, damping 30)
- **Active indicator**: `layoutId` for shared layout transitions
